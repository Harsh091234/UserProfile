## ✨ Features

✅ **User Authentication**  
- Create an account  
- Login and logout securely  

✅ **Post Management**  
- Create new posts  
- Edit your posts  
- Like and unlike posts  

✅ **Profile Customization**  
- Upload or change your profile picture
